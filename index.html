Браузерная игра "Змейка" по мотивам тетриса

Правила игры

Задача только одна - съесть как можно больше мышек, чтобы змейка была сыта. При этом нельзя, чтобы змейка сталкивалась с границами поля или собственным хвостом, иначе игра будет окончена.

Управление

  •   - Змея движется вниз.
  •   - Змея движется вверх.
  •   - Змея движется влево.
  •   - Змея движется вправо.
